# AnimalDotVitals (BLE receiver)

Receives and parses BLE vitals from the AnimalDot transmitter.

**If you see "Unable to handle compilation, expected exactly one compiler job":**  
Open **this folder** as the workspace (File → Open Folder → `AnimalDotVitals`), then build with PlatformIO or Arduino IDE. Do not build from the parent `firmware` or `sketches` folder.
